## Instalation

1. Clone the repository:

   ```
   git clone https://github.com/jakub-bartoszek/strefa-menadzera.git
   cd strefa-menadzera
   ```

2. Install dependencies:
   ```
   npm install
   ```
3. Run application:
   ```
   npm run dev
   ```
